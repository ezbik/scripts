
Reboot.macro	-   Reboot a phone, tested on Android 10
                - called by webhook /reboot
Wake.macro      - Wakes screen, can be called as /wake webhook
Ipchange.macro  - changes IP, can be called by Intent:  com.arlosoft.macrodroid.ipchange 
Autostart.macro - starts some Apps (Termux) on device start.
Capture_on.macro    - accepts ScreenCapture request , useful for Anydesk/VNC_Server
                    - called by webhook /capture_on
